# UI Enhancement Roadmap

## 1. Brand System
- Tighten brand palette with 2–3 accent hues (teal + deep blue + alert amber).
- Add a consistent AscultiCor badge style for key headers.
- Create a lightweight logo mark (simple waveform + heart) and use it in empty states.

## 2. Typography & Hierarchy
- Use a consistent display font for titles and a cleaner body font.
- Add a section title variant for panels to normalize sizing.
- Reduce visual noise by lowering subtext contrast globally.

## 3. Sidebar & Navigation
- Add active page indicator bar on the left of the active item.
- Add hover glow on active items to reinforce focus.
- Add a Quick Actions block (New Session, Add Device) with compact buttons.

## 4. Dashboard Cards
- Add subtle sparklines in stat cards (mini trend, not charts).
- Use a consistent icon badge shape (rounded-xl) across all cards.
- Add a last updated timestamp near the header.

## 5. Data Tables & Lists
- Add row hover micro-animations (translate or soft shadow).
- Add a column header style for list sections for consistent scanning.
- Use pill badges with icons for status values.

## 6. Empty States
- Replace generic icons with branded illustrations (simple SVG).
- Add a single line explaining why this matters for clinical context.
- Provide a single CTA per empty state (avoid choice overload).

## 7. Forms & Inputs
- Add input group headers (short label + hint).
- Add subtle focus glow that matches the brand color.
- Use a consistent secondary button for cancel/back.

## 8. Motion & Feedback
- Add page-load stagger for card grids (subtle, 120–180ms).
- Add saving states with inline spinners on buttons.
- Add toast styles for success/warn/error with stronger icons.

## 9. Session Detail Page
- Turn the top info block into a clinical summary card.
- Add a compact model confidence badge near prediction labels.
- Separate data sections with soft dividers or gradients.

## 10. Devices Page
- Add a health score tag (green/yellow/red).
- Add a last seen micro-indicator.
- Add a right-aligned action menu to declutter cards.
